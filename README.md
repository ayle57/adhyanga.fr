# Adhyanga.fr 🌿
Bienvenue sur le dépôt officiel du site Adhyanga.fr. Ce site, développé avec SvelteKit, a pour objectif de promouvoir les bienfaits des massages ayurvédiques et des pratiques de bien-être.

## 🚀 Technologies utilisées
- SvelteKit : Framework moderne pour le développement web.
- Vite : Outil de build rapide et performant.
- Node.js : Back-end pour le rendu SSR et API.
- TypeScript : Langage pour une base de code robuste et maintenable.

## 📂 Structure du projet
Voici une vue d'ensemble de l'arborescence principale du projet :

```txt
adhyanga.fr/
├── src/
│ ├── routes/ # Pages et endpoints
│ ├── lib/ # Composants et fonctions réutilisables
│ ├── styles/ # Fichiers CSS ou configurations Tailwind
│ └── app.html # Template HTML principal
├── static/ # Assets statiques (images, favicon, etc.)
├── package.json # Dépendances et scripts
├── svelte.config.js # Configuration SvelteKit
└── vite.config.js # Configuration Vite
```

## 🔧 Installation et lancement
### Prérequis
- Node.js (v16 ou plus récent)
- npm ou pnpm (gestionnaire de paquets)

### Étapes d'installation
1. Clonez le dépôt :
    ```bash
    git clone https://github.com/votre-utilisateur/adhyanga.fr.git
    cd adhyanga.fr
   ```
2. Installez les dépendances :
   ```bash
    npm install
   ```
3. Lancez le serveur de développement :
    ```bash
   npm run dev
   ```
4. Ouvrez votre navigateur à l'adresse suivante : http://localhost:5173

## 🛠️ Scripts disponibles
- ``npm run dev`` : Lance un serveur de développement.
- ``npm run build`` : Compile le site pour la production.
- ``npm run preview`` : Prévisualise le build de production.
## 🌐 Déploiement
Le site peut être déployé sur une plateforme compatible avec SvelteKit, comme Vercel, Netlify, ou un serveur personnalisé.
